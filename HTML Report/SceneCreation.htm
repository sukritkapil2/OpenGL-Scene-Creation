Project Description

 

This project aims to create a scene in OpenGL using the GLUT library. We have made a scene of BITS Pilani Hyderabad Campus’ lawn and the adjacent buildings. All the objects in the scene have been built from scratch using the inbuilt rendering functions of glut. Apart from that, blender objects have been imported into the scene and camera transformations are also supported.

 

Features

 

·       Translation and rotation of the scene through the keyboard – W, A, S, D can be used from rotation along the x and the y axis respectively. J, L can be used for z axis rotation. Left, right, up, down arrow keys can be used for translation along the x and y axis, whereas I, K can be used for translation along the z-axis.

 

·       Blender Objects - To incorporate blender objects, we have added a ‘bush.obj’ blender object file into the project directory. To render the same using OpenGL, we read through the obj file and draw triangular polygons using the vertices in the file. Normals are also added in the same manner.

 

Basic Building Blocks

 

For making the scene some basic 3D shapes were required –

1. Quadrilateral joining four points – For this we implemented a function drawQuad(), taking four points as parameters.

2. Cuboid with thickness h, width w, length l in the x – y plane – We used the glSolidCube() function and drew adjacent cubes to achieve the required result.

3. Cuboid with thickness w, height h, length l in the y - z plane – Similar implementation as 2.

4. Cuboid with thickness w, height h, length l in the x – z plane – Similar implementation as 2.

5. Circle of radius r – We made a function draw_circle() that uses the paramteric equation of the circle to draw the vertices.

6.  Hemisphere of radius r – We made a hemisphere using the glVertex3f() function and the parametric equation of a circle, drawing circle with decreasing radius one above the other.

7. Spiky Bush - We used cones joined with each other at an angle to achieve the same.

8. Blender Bush – We used the blender obj file to go through the polygone vertices and drew them using the OpenGL glut functions.

 

Scene Creation and Description

 

The scene of the lawn consists of these four major components –

1. Ground

Ground is at the center of the scene and consists of a rectangular base, the perpendicular pavements and a fountain at the center along with the blender imported bushes.

 

2. C Block

C Block consists of the main building, two side buildings and a stage at the center.

 

3. Library and F Block

Both have almost the same structure, consisting of one large main building, but the library has a stack of cylinders on the top.

 

4. Auditorium

Auditorium consists of a single large building.
